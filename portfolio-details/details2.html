ABC Jobs

ABC Jobs is a website for a community portal similar to LinkedIn. The portal allows users to register, login, update profiles, reset passwords, search for other users and view other users' profiles.

In this project, I used Axure to create the ABC Jobs prototype. First of all, I designed the page layout with an intuitive and attractive appearance so that users can easily interact with the site. The validation feature that I implemented on the registration form ensures that the data entered by the user is in the correct format before being saved into the system. I also use interactions in Axure to allow users to get clear and helpful feedback when performing certain actions, such as filling out forms or making adjustments to profiles.

The prototype also includes interactive elements that provide an experience similar to an actual website. Users can feel how the process of registration, login, profile updates, and browsing other users' profiles will take place in real time. With an attractive and interactive design, I hope that this prototype can help in further development and testing before complete development.
