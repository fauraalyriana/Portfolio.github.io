Know Your Neighborhood

Know your neighborhood is a website that allows users to log in using existing APIs and the system will retrieve basic information such as email names from the API.

In creating the "Know Your Neighborhood" website, I chose Visual Studio Code as the main code editor tool to develop the entire project. To develop the backend, I use Spring Tool Suite (STS) as the right choice. Then, to access the API and retrieve basic information such as name and email from the user, I use Postman. With Postman, I can make HTTP requests to existing APIs and see the results, thus ensuring the connection and integration with the API is working properly. Also, in front end development, I use React.js as an option for creating user interfaces. Using React.js, I was able to create reusable UI components, making frontend development and maintenance easier.

The "Know Your Neighborhood" project is a collaboration that combines advanced technologies to produce a functional and user-friendly website. Visual Studio Code, Spring Tool Suite, and Postman were instrumental in solid backend development and access to reliable APIs, while React.js was used to create attractive and interactive user interfaces.
